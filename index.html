Basic HTML Tags for Dummies

Tag Name Description
<p> Paragraph Tag This tag formats any text between the opening tag and the closing tag as a standard paragraph or main body text.
<h2> Heading Tag Using this tag will format any text between the opening tag and the closing tag as a Heading 2 (a type of subheading.)
<b> Bold Tag This tag will format any text between the opening tag and the closing tag as bold.
<i> Italic Tag This tag will format any text between the opening tag and the closing tag as italic.
<u> Underline Tag This tag will format any text between the opening tag and the closing tag as underlined.
<img> Image Tag In HTML, the element used to display images is defined with a source attribute specifying the path to the image file and an alternative text attribute providing a description.
